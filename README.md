# simpleweb-base

**only for personal use**

simple baseline for prototyping at web frontend.

based on https://webpack.js.org/guides/


## example codes

- p5js_example.js (http://p5js.org/)

- three_example.js (https://threejs.org/)

### how to use
edit `webpack.config.js`

```
entry: './src/main.js',
```

## development start

```
$ npm install
$ npm start
```

## build

```
$ npm run build
```
